A no-code recommendation engine that helps visitors find the right programme through guided forms and conditional rules.

== Description ==

Smart Programme Finder lets you build recommendation forms, define conditional rules, and display dynamic results — all without writing code.

**Features:**

* Visual form builder in wp-admin
* Rule-based recommendation engine
* AJAX form submission (no page reload)
* Modal result popup
* Shortcode support: `[spf_form id="1"]`
* Elementor widget
* Mobile responsive
* Beginner-friendly admin UI

== Installation ==

1. Upload the `smart-programme-finder` folder to `/wp-content/plugins/`.
2. Activate the plugin through the Plugins screen.
3. Navigate to **Programme Finder** in the admin sidebar.
4. Create fields in the Form Builder.
5. Add rules in the Rule Builder.
6. Embed with the shortcode `[spf_form id="1"]` or the Elementor widget.
