## 1. Executive Summary

Smart Programme Finder is a WordPress plugin designed to help schools, academies, institutes, and training providers guide website visitors toward the right programme through a simple interactive recommendation experience.

Instead of asking visitors to browse multiple programme pages manually, the plugin allows an organization to create a short guided form that collects a few answers and instantly returns a relevant recommendation. The experience is fast, modern, and designed to feel more like a smart advisor than a traditional web form.

---

## 2. The Problem It Solves

Many education websites present visitors with too many programme choices and too little guidance. As a result:

- Prospective students are unsure where to start.
- Website visitors abandon the journey before finding a suitable course.
- Enquiry teams receive lower-quality or less-informed leads.
- Marketing teams struggle to create interactive recommendation experiences without custom development.

Smart Programme Finder addresses this gap by giving site owners a no-code way to turn programme discovery into a guided recommendation journey.

## 4. What the Product Does

Smart Programme Finder enables website owners to:

- Build custom recommendation forms without coding.
- Add question fields such as dropdowns, radio choices, and checkbox groups.
- Define rules that map user answers to programme recommendations.
- Show recommendation results instantly using AJAX, without page reload.
- Display the recommendation inside a clean modal popup.
- Embed the experience via shortcode or Elementor widget.

The result is a smoother path from visitor uncertainty to programme discovery.

---

## 5. Core Product Capabilities

### No-Code Form Builder

- Create forms directly inside WordPress.
- Add fields through a guided interface.
- Configure labels, options, and required fields.

### Recommendation Rules

- Define logic that connects answers to outcomes.
- Control which recommendation appears for each answer path.
- Add a fallback result for unmatched responses.

### Instant Results

- Submit without refreshing the page.
- Display tailored results immediately.
- Give users a clear next step through a CTA button.

### Flexible Embedding

- Place forms anywhere using shortcode.
- Add forms to custom layouts with an Elementor widget.

## 6. User Experience Overview

### Administrator Experience

1. Create a new form in WordPress.
2. Add a small number of questions.
3. Define recommendation rules.
4. Preview the experience.
5. Publish and embed the form.

The admin experience is designed to be beginner-friendly, guided, and fast to learn.

### Visitor Experience

1. A visitor answers a short set of questions.
2. The form submits instantly.
3. A recommendation appears in a modal popup.
4. The visitor follows a CTA to learn more or apply.

The frontend experience is designed to feel clear, lightweight, and modern.

## 8. MVP Deliverable Scope

The first release is intentionally focused.

Included in MVP:

- Basic form builder.
- Single-condition rule builder.
- AJAX submission.
- Modal recommendation result.
- Shortcode embedding.
- Elementor widget.

Not included in MVP:

- Lead storage.
- CRM or email integrations.
- Multi-condition advanced logic.
- Analytics dashboard.
- Scoring-based recommendations.

This keeps the first version practical, clear, and commercially usable without overcomplicating delivery.

---

## 9. Why This Product Is Differentiated

Smart Programme Finder is positioned differently from generic form builders.

- It is recommendation-led rather than submission-led.
- It is designed specifically for choice-heavy organizations such as schools and institutes.
- It combines guided logic, dynamic results, and easy embedding in one WordPress-native workflow.
- It is built to feel like a decision engine, not a simple contact form.
